# 📚 Knowledge Base — Metodo Sincro

Metti qui i tuoi documenti. Antigravity li legge e li distribuisce agli agenti.

## Cartelle

| Cartella | Cosa metterci |
|---|---|
| `brand/` | Chi siete, mission, valori, tone of voice, posizionamento |
| `customer-avatar/` | Avatar cliente ideale, pain points, desideri, obiezioni |
| `creative-strategy/` | Il documento su come creare le creative che convertono |
| `ads-guidelines/` | Best practices Meta, Andromeda, regole per le campagne |
| `landing-pages/` | Copy vincente, varianti A/B, screenshot di riferimento |
| `video-templates/` | Script, storyboard, template video |
| `agent-playbooks/` | Playbook generati automaticamente per ogni agente |

## Formati supportati
- PDF, DOC, DOCX, TXT, MD
- **HTML** (pagine web salvate, email, landing page)
- Immagini (PNG, JPG)
- Link (salvali in un file .txt)

## Come funziona
1. Metti i file nella cartella giusta
2. Dì ad Antigravity "leggi i documenti nella knowledge base"
3. Antigravity crea i playbook per gli agenti
